# **App Name**: Nomad Cribs

## Core Features:

- Secure User Authentication: User authentication with secure signup and login functionality and password validation: Password must contain at least 6 characters.
- Profile Management: Profile management: Users can create, edit, and manage their profiles, with options for adding profile pictures, names, ages, emails, and phone numbers.
- Browse Listings: Listing Display: Show a scrollable list of hostel or roommate ads, each including a thumbnail, title, location, price, and short description.
- Listing Details: Listing details: a detail view with all available information. Include image carousels, detailed descriptions, addresses with map integration, facilities, rental information and contact information for the owners.
- Filter and Sort Options: Interactive filtering system that users can use to select preferences about pricing and amenity options to refine search
- Add to Favorites: User favorites: Enables users to add hostels and roommate postings to favorites and later retrieve them easily
- Roommate Matching: Roommate matching: a tool that suggests potential roommates based on user provided criteria and LLM analysis of listing descriptions.

## Style Guidelines:

- Primary color: A saturated, but not vibrant, blue (#6495ED). It suggests trustworthiness, openness, and stability, aligning with the need for safety and reliability.
- Background color: Light blue (#E6EAF7), approximately the same hue as the primary color, but desaturated to about 20% to provide a calming and unobtrusive backdrop.
- Accent color: A lighter blue (#7EB5FF), created by shifting the primary color’s hue approximately 30 degrees and increasing the brightness, offering visual contrast to the primary and background colors.
- Body text: 'PT Sans' for a warm and modern look that works well in various sizes, including paragraphs and headings. The body has blocks of text longer than short paragraphs.
- Headlines: 'Playfair' (serif). Note: currently only Google Fonts are supported. It is suitable for headlines or small amounts of text. 'PT Sans' will be the font for the body of the page.
- Minimal and modern icons, using the primary color. Consistent style across the app.
- Clean, well-spaced layouts. Cards with rounded corners and soft shadows for a modern, mobile-friendly design.
- Subtle transitions and animations for loading states and UI interactions. Smooth fade-in animations for new content.
